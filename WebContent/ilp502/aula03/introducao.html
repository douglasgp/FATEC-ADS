Aula 03 - CSS


  • Flexibilidade ao HTML
  • Padronização das páginas
  • Facilidade de manutenção


Assim como o HTML, o CSS não é realmente uma linguagem de programação. Também não é uma linguagem de marcação (markup), mas sim uma linguagem de folhas de estilos.

Isso quer dizer que o CSS permite que você aplique estilos seletivamente a elementos em documentos HTML.

    CSS Inline
  • Toda configuração de estilos CSS são inseridos diretamente dentro de cada tab utilizada na página HTML. É recomendado utilizar esta forma em projetos com páginas simples, pois isso demandará muito trabalho.
    CSS Interno:
  • É implementado toda configuração CSS antes de terminar a tag head, informando quais as características de cada uma delas.
    Arquivo ".css"
  • Esse tipo de arquivo recebe apenas configurações CSS, e não permite código HTML .Toda página html que optar pelas confugurações deste arquivo, é necessário apenas invocá-lo.
    Seletor
  • Referência a um único elemento na página (id). Para empregá-lo utiliza-se a cerquilha ( # ) dentro do arquivo '.css'.
    Classe
  • Referência a um ou a m grupo de elementos na página (class). Para usá-lo, aplicase a instrução ponto ( . ) dentro do arquivo '.css'.
