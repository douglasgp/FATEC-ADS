Introdução ao CSS

O Cascading Style Sheets é uma linguagem utilizada para definir a apresentação (aparência de documentos que adotam para o seu desenvolvimento marcação (como XML, HTML, e XHTML e etc...). O CSS define como serão exibidos os elementos contidos no código de um documento e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.
